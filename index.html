This page demonstrates how CSS selectors work.

id

ids can be applied to only one item on a page.

Here's how we hook an id attribute to an element:

id="id-div"

Here's how the CSS is applied to the element:

div#id-div{
    background-color:yellow;
}

Inside my-div
class

classes can be applied to more than one item on a page.

Here's how we hook a class attribute to an element:

class="my-list"

Here's how the CSS is applied to the element:

li.my-list{
    background-color:orange;
}

compound

A compound selector allows multiple selectors to accept the same rules.

Here's how the CSS is applied to the elements:

li.new-list,div#new-div{
    background-color:blue;
}

child

A child selector allows us to access the next immediate elements inside the current element.

Here's how we hook a class attribute to the parents of an element:

class="my-kids"

Here's how the CSS is applied to the element:

ul.my-kids > li{
    background-color:green;
}

Here's what it looks like:

  • I'm in an li inside a ul with a class of my-kids
  • I'm in an li inside a ul with a class of my-kids
  • I'm in an li inside a ul with a class of my-kids
descendent

Using a descendent selector, we can access any element at any depth inside the current element.

Here's how we hook a class attribute to the grandparent of an element:

class="grandkids"

Here's how the CSS is applied to the element:

div.grandkids li{
    background-color:red;
}

Here's what it looks like:

  • I'm in an li inside a div with a class of grandkids
  • I'm in an li inside a div with a class of grandkids
  • I'm in an li inside a div with a class of grandkids
multiple

We can apply more than one class to any item on our page.

Here's how we hook multiple classes to an element:

class="add-border add-background"

Here's how the CSS is applied to the element:

.add-border{
    border:1px solid #000;
}
